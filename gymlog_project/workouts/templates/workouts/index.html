Welcome to Your Gym Log!

This is the Django version of your gym tracking application.

Features:

  • Log your workouts with exercises, sets, reps, and weights
  • Track your progress over time
  • View workout history
  • Track body metrics (weight, height)
  • Calendar view of your workouts

Django Admin Interface:

Django comes with a powerful admin interface where you can manage all your data:

Go to Admin Panel

Note: You'll need to create a superuser first (see instructions)

API Endpoints:

  • GET /api/workouts/ - List all workouts
  • GET /api/exercise_names/ - List all exercise names
  • GET /api/body_metrics/ - List all body metrics
  • GET /api/calendar_data/ - Get calendar data
